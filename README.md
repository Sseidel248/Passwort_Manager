KiiTree

Kiitree ist ein Programm zur Speicherung von Passwörtern in einer selbst erzeugte Stand-Alone Datenbank, diese befindet sich entweder auf den lokalen Rechner oder aus einem Server. Wobei der Speicher- und Ladepfad selbst gewählt werden kann. 
Benutzerdaten können gespeichert, bearbeitet und die Zwischenablage gepackt werden. Dadurch braucht der Benutzer sich nur sein Masterpasswort merken. Das Tool kann einem auch die Passwörter bei Bedarf ausdrucken und prüft die Passwörter hinsichtlich ihrer Stärke.
